Comment connecter l'API Instagram ?

L'API Graph donne les vraies stats de ton compte (reach, impressions, demographics).

  1. Passe @samourais_ en compte Creator ou Business (Settings > Account type)
  2. Cree une Facebook App (type Business)
  3. Ajoute le produit Instagram Graph API
  4. Lie ta Page Facebook au compte IG
  5. Dans Graph API Explorer, genere un token avec les permissions: instagram_basic, instagram_manage_insights, pages_show_list, pages_read_engagement
  6. Colle le token ci-dessous et clique Configurer

Pourquoi utiliser un proxy ?

Instagram et TikTok bloquent les IP de serveurs cloud (comme Railway).

Un proxy residentiel fait croire que la requete vient d'une vraie connexion internet.
Services recommandes : Webshare, Smartproxy, Bright Data

Format: http://username:password@host:port

Sessions (Cookies)

Comment recuperer ses cookies ?

Les cookies permettent au scraper de se connecter comme toi sur les plateformes.

  1. Installe Cookie Editor (Chrome)
  2. Connecte-toi sur la plateforme (ex: instagram.com)
  3. Ouvre Cookie Editor → Export → JSON
  4. Colle dans un fichier .json
  5. Upload ci-dessous
Les cookies expirent au bout de quelques semaines. Refais la procedure si le scraper bloque.
